YouTube字幕・文字起こしツールの使い方

このツールは、YouTube動画の字幕取得や音声からの文字起こしに必要なコマンドを生成します。

事前準備（ローカルPC）:

1. yt-dlp をインストール:
pip install yt-dlp または brew install yt-dlp

2. Whisper をインストール（文字起こし用）:
pip install openai-whisper

使い方:

1. 字幕取得（字幕あり動画）
動画に字幕がある場合、yt-dlpで字幕ファイルを直接ダウンロードします。自動生成字幕・手動字幕のどちらにも対応。

2. 音声→文字起こし（字幕なし）
字幕がない動画の場合、まず音声をMP3でダウンロードし、Whisperで文字起こしします。

3. 保存済み
取得した文字起こし結果をアプリ内に保存・管理できます。
